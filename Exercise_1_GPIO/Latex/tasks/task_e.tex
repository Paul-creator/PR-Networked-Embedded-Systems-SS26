\textbf{Assignment of the values to the 7-segment display:}

Each decimal digit is represented by a unique combination of the seven segments A to G. A value of \texttt{1} means that the corresponding segment is turned on, while a value of \texttt{0} means that it remains off. Since the display is connected to pins PJ0 to PJ6, the segment pattern can be assigned directly to the GPIO output bits.

A suitable approach is to store the segment combinations for all digits from 0 to 9 in an array. Each array entry contains the bit pattern for one digit. The program then selects the corresponding pattern depending on the current step and writes it to the output data register of Port J using bit manipulation.

This method avoids setting every segment individually for each number and improves reusability. It also satisfies the requirement that the display functionality should be implemented as a reusable method.
